# Viewmodel.js
Viewmodel.js is a lightweight script used to adjust and fine-tune weapon position in-game.  
It is designed to work with userscript based sources and allows manual control over X, Y, and Z offsets.

<br>

Settings

```js
X: 0.45
Y: -0.25
Z: 0.32
```
